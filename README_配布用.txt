================================================================================
                    会計データ可視化ダッシュボード
                         バージョン 1.0.0
================================================================================

【概要】
--------
このアプリケーションは、会計データを可視化・分析するためのデスクトップツールです。
月次推移、セグメント分析、取引先分析などの機能を提供します。

【動作環境】
------------
- Windows 10 / Windows 11
- 画面解像度: 1280x800 以上推奨

【起動方法】
------------
1. 「会計ダッシュボード.exe」をダブルクリックして起動します
2. 初回起動時はセキュリティ警告が表示される場合があります
   「詳細情報」→「実行」をクリックしてください

【使い方】
----------

■ フィルタパネル（左側）
  - 年度: 表示したい年度をチェックボックスで選択（複数選択可）
  - セグメント: 表示したいセグメントを選択（複数選択可）
  - 科目: 売上高、売上原価などから選択

■ ダッシュボードタブ
  - サマリー: 合計金額、前年比、取引先数などの主要指標
  - 月次推移グラフ: 年度別の月次推移を折れ線グラフで表示
  - セグメント構成比: 円グラフでセグメント別の構成比を表示
  - その他グラフ: 棒グラフなど

■ 取引先分析タブ
  - 取引先ランキング: 売上高上位の取引先を一覧表示
  - 業種別構成比: 業種ごとの構成比を円グラフで表示
  - 取引先別推移: 取引先を選択すると月次推移を表示

■ セグメント分析タブ
  - セグメント比較表: 当期・前期の比較、前年比、構成比を表示
  - セグメント構成比グラフ: 円グラフで表示
  - 年度別比較グラフ: 選択した全年度のセグメント別実績を比較
  - 月次推移グラフ: セグメントを選択すると月次推移を表示

■ CSVインポート
  - ツールバーの「CSVインポート」ボタンをクリック
  - CSVファイルを選択
  - ワイド形式（1月〜12月の列を持つ形式）に対応
  - 「追加」または「置換」を選択してインポート

■ CSVエクスポート
  - ツールバーの「CSVエクスポート」ボタンをクリック
  - 保存先を指定してエクスポート
  - 現在のフィルタ条件に基づいたデータが出力されます

【データファイル】
------------------
- データベースファイル: data\accounting.db
- このファイルに会計データが格納されています
- CSVインポートでデータを追加・置換できます

【CSVインポート形式】
--------------------
以下の形式のCSVファイルをインポートできます：

■ ワイド形式（推奨）
年度,開示セグメント名称,事業部名称,部門コード,部門名称,取引先コード,取引先名称,WITC業種名①,昇順,科目名称,1月,2月,3月,...,12月,合計

■ ロング形式
year,month,segment,division,dept_code,dept_name,customer_code,customer_name,industry,account,amount

※日本語カラム名（年度、セグメント等）も対応しています

【トラブルシューティング】
--------------------------

Q: アプリが起動しない
A: 以下を確認してください
   - Windowsが最新の状態か確認
   - セキュリティソフトがブロックしていないか確認
   - 管理者として実行を試す

Q: グラフが表示されない
A: 以下を確認してください
   - フィルタで年度・セグメントが選択されているか
   - データベースにデータが存在するか

Q: 日本語が文字化けする
A: フォントの問題の可能性があります
   - システムに「MS Gothic」または「Meiryo」フォントがインストールされているか確認

Q: CSVインポートに失敗する
A: 以下を確認してください
   - CSVファイルのエンコーディング（UTF-8, Shift-JIS対応）
   - 必須カラムが含まれているか
   - 数値データに不正な文字がないか
